Tarjan 强连通分量学习笔记

发表于 2018-03-27 | 分类于 Algorithm

在一个有向图中，如果某两点间都有互相到达的路径，那么称中两个点强连通，如果任意两点都强连通，那么称这个图为强连通图；一个有向图的极大强连通子图（不被原图其它强连通子图包含）称为强连通分量。

Tarjan 算法可以在 的时间内求出一个图的所有强连通分量。

若将有向图的强连通分量都视作一个点，则原图会形成有向无环图（DAG）。

定义

栈，DFS 树中同一棵子树中的点在栈中是相邻的。

表示进入节点 时的时间（时间截）。

表示由节点 开始搜索所能到达的点中，在搜索树上是 的祖先且 最小的节点的 。

描述

  1. 选定一个节点作为根节点，开始 DFS；
  2. 初始化当前点的 和 均为当前时间截，并进栈；
  3. 遍历当前点 的所有邻接点（ 出边连的点）；
  4. 如果某个邻接点 在栈中，更新 ；
  5. 如果某个邻接点 不在栈中，则对 进行 DFS，完成后更新 ；
  6. 所有邻接点都完成 DFS 后，如果满足 ，则将栈中从 到栈顶的所有元素出栈，并标记为一个强连通分量。

解释

如果某个邻接点 在栈中，更新 ；

已被访问过且还未出栈，说明 找到它的祖先 ，并形成了一个环，此时要用 去更新 的最远祖先。

如果某个邻接点 不在栈中，则对 进行 DFS，完成后更新 ；

点 出发能到达的最远祖先，点 一定也能到达。

所有邻接点都完成 DFS 后，如果满足 ，则将栈中从 到栈顶的所有元素出栈，并标记为一个强连通分量。

如果当前点 为根的子树下，无论怎么走也无法到达 的祖先，说明整棵子树下的点组成一个强连通分量。因为从 进栈后，所有进栈的点都是子树内的点，所以这个点在栈中的位置到栈顶位置中的点为同一个强连通分量。
